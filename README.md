# SPA groep project Matthijs Bossier, Koen Damme, Dogan Atak, Devin Braat klas A


## Opstarten

SPA-angular:

- npm install
- ng serve

SPA-server: 
- npm install
- npm start

Lokaal draait de angular-kant van de applicatie op `http://localhost:4200/`.

Lokaal draait de server-kant (api)
`http://localhost:3000/`
